# Tecnicatura universitaria en Programación - UTN.BA | e-Learning

![UTN.BA](assets/img/TuP-background.webp)

## 📄 Descripción
Sitio web dedicado a recopilar, centralizar y compartir recursos, apuntes, ejercicios y materiales de la [Tecnicatura universitaria en Programación](https://sceu.frba.utn.edu.ar/e-learning/detalle/carrera/3906/tecnicatura-universitaria-en-programacion?id=184176) de la [UTN.BA | e-Learning](https://sceu.frba.utn.edu.ar/e-learning/). El objetivo es facilitar el acceso a información relevante tanto para los estudiantes de la carrera como para cualquier persona interesada en la misma.
