# tech-stack.md

Tecnologías permitidas, recomendadas y restringidas para este proyecto.
No describe al desarrollador — describe el proyecto.

---

## Tecnologías principales

- **HTML5** — semántico y accesible
- **CSS3** — vanilla, sin frameworks
- **JavaScript** — vanilla ES6+, sin frameworks ni bundlers

## Herramientas

- **Git** — control de versiones
- **GitHub** — repositorio remoto
- **GitHub Pages** — deploy
- **VSCode** — editor

## Recursos externos permitidos

- **Google Fonts** — tipografías vía CDN
- **Material Symbols Outlined** — iconos vía CDN

## Principios

- Sin frameworks CSS (Bootstrap, Tailwind, etc.).
- Sin frameworks JS (React, Vue, etc.).
- Sin bundlers (Webpack, Vite, etc.).
- Sin npm ni `package.json`.
- Priorizar tecnologías nativas del navegador.
- Librerías externas solo si aportan algo que no se puede lograr razonablemente con código propio — y deben justificarse antes de agregarse.
- Priorizar simplicidad y mantenibilidad sobre sofisticación técnica.

## Escalación

Si una tarea requiere una tecnología no incluida en este archivo:

1. Explicar por qué sería necesaria.
2. Proponer alternativas compatibles con el stack actual.
3. Esperar confirmación antes de incorporarla.
